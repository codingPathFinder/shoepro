## 💻 주요 기능
1. SMS 인증을 통한 회원가입, 이메일 찾기, 비밀번호 찾기 
2. 장바구니에 상품과 서비스를 각각 담아서 관리하기
3. 신용카드 혹은 계좌이체로 결제하기
4. 포인트 적립, 쿠폰 사용하기
5. 주문 취소하기
6. 배송 조회하기
7. 리뷰 작성 및 상품 문의 작성하기
